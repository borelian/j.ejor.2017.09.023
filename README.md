Fourth and following lines: Cost of each client to each location:   
Fourth line: <cost_client1_loc1> <cost_client2_loc1> ... <cost_clientN_loc1>
Fifth line:  <cost_client1_loc2> <cost_client2_loc2> ... <cost_clientN_loc2>
...
